Estimating date of composition

This was a research project I worked on under Professor David Bamman at the UC Berkeley School of Information. Our goal was to estimate the date of composition of older books, where books published posthumously might mean the composition date was significantly different from the publication date. I created features for a machine learning model to do so. My work contributed to a publication in the Joint Conference on Digital Libraries 2017.

Twitter Topic Analysis for NIH

I joined a research project under EECS Professor Laurent El Ghaoui where I used sparse machine learning techniques to analyze changes in how people talked about public health announcements on Twitter.

Topic Analyses

This studies trending topics in New York Times articles using different algorithmic techniques in the R programming language.

Fruitful

Fruitful is a web app that teaches simple self-exams for breast, testicular, and skin cancers through colorful fruit animations. This won 1st place in the 2015 Women in STEM hackathon.

Breadth-Major Overlaps

This uses BeautifulSoup (Python) to parse through UC Berkeley webpages to see which classes required for majors/minors can also be used to fulfill breadth requirements.

Network

I designed an AI program in Java to play the game Network using the minimax algorithm and alpha-beta pruning.

Depth Perception Simulator

This project, written in C, can distinguish between far away and close by objects by creating a depth map with respect to each pixel, and then normalizing that map.
